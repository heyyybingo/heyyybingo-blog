# dns 解析

## 作用

DNS( Domain Name System)是“域名系统”的英文缩写，是一种组织成域层次结构的计算机和网络服务命名系统，它用于 TCP/IP 网络，它所提供的服务是用来将主机名和域名转换为 IP 地址的工作。

## 解析过程

从客户机到本地 dns 服务器之间的查询为递归查询
dns 服务器之间的查询为迭代查询

## 缓存

- 浏览器缓存
- 系统缓存
- 路由器缓存
- ISP dns 缓存（比如你用的是电信的网络，则会进入电信的 DNS 缓存服务器中进行查找。）
